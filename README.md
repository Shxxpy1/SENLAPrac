# Итерация 6. Асинхронный JS

Практическое задание:
Изучить методы REST и promise. Добавить загрузку репозиториев из API GitHub. При клике на репозиторий (элемент списка) происходит переход на страницу с подробным описанием репозитория (предусмотреть кнопку перехода назад). Так же добавить изменение ссылки в адресной строке в соответствии с открытой страницей:
1. / - список репозиториев
2. /?username=<значение из поля ввода имени пользователя>
3. /<:id репозитория> - страница подробного просмотра репозитория
